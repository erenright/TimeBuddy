End of Day Pause

At the end of your working day, TimeBuddy will automatically pause the time tracking timer. This is done to prevent accumulating time when you have gone home for the day and forgotten to pause the timer or quit the program.

The default end of work day is 5pm. This can be changed by doing the following:

  1. Open the Settings window
  2. Select the Options tab

Make your desired changes and close the window to save.

The Start of day setting is not currently used.
